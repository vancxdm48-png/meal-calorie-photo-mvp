# Meal Calorie Photo MVP

写真1枚に写った料理をもとに、食事全体のカロリーを概算するMVPです。

## 現在の範囲

- 写真選択とプレビュー
- APIキーなしで動くデモ推定
- 料理ごとの量補正
- マヨネーズ、油、バター、ドレッシングなどの隠れカロリー追加
- 成人男性・成人女性の1日摂取目安との比較
- 再計算後の合計カロリー範囲表示
- 最近の食事記録をこの端末内に保存
- スマホのホーム画面に追加できるPWA設定
- 医療・栄養指導ではないことの免責表示

## スマホで使う

開発中はPCとスマホを同じWi-Fiにつないで、PCのローカルIPアドレスにアクセスします。

```bash
npm run dev
```

起動後に表示される `Network` のURL、またはPCのIPアドレスを使ってスマホで開きます。

```text
http://PCのIPアドレス:5173/
```

iPhoneではSafariで開き、共有メニューから「ホーム画面に追加」を選ぶとアプリのように起動できます。AndroidではChromeのメニューから「ホーム画面に追加」または「アプリをインストール」を選びます。

本番でPWAとして安定してインストール、オフライン利用するにはHTTPS配信が必要です。ローカルWi-Fiでは、まずスマホで撮影、写真選択、推定、記録保存の動作確認を行う用途に向いています。

## 基準データ

1日の摂取目安は、厚生労働省「日本人の食事摂取基準（2025年版）」の推定エネルギー必要量を参照しています。

- 性別: 成人男性 / 成人女性
- 年齢: 18-29歳、30-49歳、50-64歳、65-74歳、75歳以上
- 身体活動レベル: 低い / ふつう / 高い

## 次に実AIへつなぐ場合の入力例

```json
{
  "task": "estimate_calories_from_food_photo",
  "locale": "ja-JP",
  "image": {
    "type": "base64_or_url",
    "value": "uploaded_meal_photo"
  },
  "user_hint": "ご飯は普通盛り。みそ汁も写っています。",
  "requirements": {
    "detect_multiple_items": true,
    "return_calorie_range": true,
    "separate_fact_and_assumption": true
  }
}
```

## 注意

写真だけでは正確な重量、調理油、ソース、隠れた具材を判定できません。表示値は概算であり、医療、減量、糖尿病管理、栄養指導などの判断には使用しない前提です。
